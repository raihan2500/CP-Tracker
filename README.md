# CP Tracker - Competitive Programming Companion

Modern dark-themed React Native (Expo) UI with:

- Bottom tabs: Home, Contests, Practice, Notes, Profile
- Card-based minimal design system
- Dashboard stats, heatmap, and rating chart
- Contest filters and reminder CTA
- Practice queue and bookmarks
- Notes search and editor
- Profile summary, leaderboard, and comparison graph

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm run start
```

Then press `a` (Android), `i` (iOS on macOS), or open with Expo Go.
